# Glossary

## Business Terms

### Auction Terms

| Term | Definition | Malay Translation |
|------|------------|-------------------|
| **Auction** | A public sale where items are sold to the highest bidder | Lelong |
| **Bid** | An offer to buy at a specific price | Tawaran |
| **Starting Bid** | The minimum price at which bidding begins | Harga Mula |
| **Reserve Price** | The minimum price the seller will accept | Harga Simpanan |
| **Buy Now Price** | Fixed price to purchase immediately without bidding | Beli Sekarang |
| **Current Bid** | The highest bid at any given moment | Tawaran Tertinggi |
| **Bid Increment** | Minimum amount by which each bid must increase | Kenaikan Tawaran |
| **Winning Bid** | The highest bid when the auction ends | Tawaran Menang |
| **Outbid** | When someone places a higher bid than yours | Tawaran Dilepasi |
| **Auto-bid / Proxy Bid** | Automatic bidding up to a maximum amount | Tawaran Auto |
| **Auction House / Seller** | Person or entity conducting the auction | Penjual |
| **Bidder / Buyer** | Person placing bids | Pembeli |

### E-commerce Terms

| Term | Definition | Malay Translation |
|------|------------|-------------------|
| **Product** | Item available for sale | Produk |
| **SKU** | Stock Keeping Unit - unique product identifier | Kod SKU |
| **Inventory** | Quantity of products in stock | Inventori |
| **Out of Stock** | No items available for purchase | Stok Habis |
| **Cart** | Temporary holding for items before purchase | Troli |
| **Checkout** | Process of completing a purchase | Pembayaran |
| **Order** | A confirmed purchase request | Pesanan |
| **Subtotal** | Sum of items before tax/shipping | Jumlah Separuh |
| **Grand Total** | Final amount including all charges | Jumlah Keseluruhan |
| **Shipping** | Delivery of purchased items | Penghantaran |
| **Tracking Number** | Code to monitor shipment status | Nombor Penjejakan |
| **Return** | Sending back purchased items | Pulangan |
| **Refund** | Money returned for cancelled/returned orders | Bayaran Balik |

### Live Streaming Terms

| Term | Definition | Malay Translation |
|------|------------|-------------------|
| **Live Stream** | Real-time video broadcast | Siaran Langsung |
| **Streamer / Host** | Person broadcasting the stream | Penyiar |
| **Viewer** | Person watching the stream | Penonton |
| **Go Live** | Start broadcasting | Mula Siaran |
| **Stream Room** | Virtual space where stream takes place | Bilik Siaran |
| **Chat** | Real-time text communication during stream | Sembang |
| **Moderator** | Person managing chat and stream | Moderator |
| **Screen Share** | Broadcasting computer screen | Kongsi Skrin |
| **Recording** | Saved copy of live stream | Rakaman |
| **Replay** | Watching recorded stream later | Ulangan |

### User Roles

| Term | Definition | Malay Translation |
|------|------------|-------------------|
| **Buyer** | User who purchases items | Pembeli |
| **Seller** | User who lists and sells items | Penjual |
| **Admin** | Platform administrator | Pentadbir |
| **Guest** | Non-registered visitor | Tetamu |
| **Verified Seller** | Seller with confirmed identity | Penjual Disahkan |
| **Follower** | User subscribed to seller updates | Pengikut |

### Payment Terms

| Term | Definition | Malay Translation |
|------|------------|-------------------|
| **FPX** | Financial Process Exchange - Malaysian online banking | FPX |
| **E-wallet** | Digital payment app (GrabPay, Touch 'n Go) | Dompet Digital |
| **COD** | Cash on Delivery - pay when item arrives | Bayaran Semasa Terima |
| **Payment Gateway** | Service processing online payments | Gerbang Pembayaran |
| **Webhook** | Automated message sent when event occurs | Webhook |
| **Settlement** | Transfer of funds to seller | Penyelesaian |

## Technical Terms

### Architecture

| Term | Definition |
|------|------------|
| **API** | Application Programming Interface - how systems communicate |
| **REST** | Representational State Transfer - API architecture style |
| **WebSocket** | Protocol for real-time bidirectional communication |
| **JWT** | JSON Web Token - secure authentication token |
| **Database** | Structured storage for application data |
| **Cache** | Temporary fast storage for frequently accessed data |
| **CDN** | Content Delivery Network - distributed static file servers |
| **Webhook** | HTTP callback triggered by events |

### Frontend

| Term | Definition |
|------|------------|
| **Component** | Reusable UI building block |
| **State** | Data that changes over time in the app |
| **Props** | Data passed from parent to child component |
| **Hook** | Function to use React features in components |
| **SSR** | Server-Side Rendering - pages rendered on server |
| **CSR** | Client-Side Rendering - pages rendered in browser |
| **Hydration** | Process of making SSR page interactive |

### Backend

| Term | Definition |
|------|------------|
| **Endpoint** | URL where API requests are sent |
| **Middleware** | Code that runs before request handler |
| **Repository** | Data access layer |
| **Service** | Business logic layer |
| **Handler** | HTTP request processor |
| **Migration** | Database schema version control |
| **Rate Limiting** | Controlling request frequency |

### Mobile

| Term | Definition |
|------|------------|
| **Native** | Built specifically for one platform (iOS/Android) |
| **Cross-platform** | Works on multiple platforms |
| **APK** | Android Package - Android app file |
| **IPA** | iOS App Store Package - iOS app file |
| **Push Notification** | Message sent to device from server |
| **Deep Link** | URL that opens specific app screen |
